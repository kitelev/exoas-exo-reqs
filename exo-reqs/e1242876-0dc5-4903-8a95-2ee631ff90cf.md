# req(exo): the effort list sorts tasks with a specific start-time before tasks without a time

## Job Story

When I look at a list of efforts, I want the ones I scheduled to a specific time of day to appear before the unscheduled ones, so my time-blocked work surfaces at the top and untimed items fall to the end.

## Statement (Gherkin)

```gherkin
Given a task A with a specific start time (e.g. 09:00:00)
  And a task B with no time of day (00:00:00 / null)
When the effort list is sorted by start time
Then task A is ordered before task B
  And two tasks both with a time are ordered ascending by time of day
```
