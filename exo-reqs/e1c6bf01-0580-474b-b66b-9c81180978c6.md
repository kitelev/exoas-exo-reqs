# req(exo): ExoSync keep-local consolidation of MULTIPLE cross-path stragglers converges in ONE sync

## Job Story

When I keep my local version for SEVERAL conflicts at once (a bulk co-location migration left N stragglers), I want a single sync to resolve ALL of them — so I don't have to Sync N times (one straggler healed per sync) and watch the rest re-appear in quarantine every cycle.

## Statement (Gherkin)

```gherkin
Given N (>1) keep-local resolutions are queued in the deferred-push outbox for one AssetSpace
And each is a cross-path (co-located rename) conflict whose stale sibling still holds the based-on remote blob
When the next sync flushes the outbox
Then ALL N chosen (local) contents are pushed and ALL N stale sibling copies are deleted in ONE atomic commit
And no queued resolution is deferred with "head moved" within the batch
And the outbox is fully drained
And a subsequent sync re-derives NO conflict for any of them (keep-local sticks for every straggler in one sync)

Given N keep-local resolutions are queued
And a genuine concurrent EXTERNAL writer moves the AssetSpace head between the flush's head fetch and its commit
When the next sync flushes the outbox
Then the WHOLE batch is aborted by the expectedHeadSha CAS (nothing overwritten)
And every queued resolution stays in the outbox to retry next sync (zero data loss)
```
